## Context

Work has an internal `dxctl` ‚Äî a charmbracelet TUI over common dev-experience CLIs (PRs, tmux, k8s). Work IP, not open-sourced. The *idea* is replicated on the personal M5 (headless dev box driven over SSH from desktop/Air **and** iPhone via Termius).

First module is **tmux**. `forgectl tmux` supersedes the bash `s` script; `sesh` stays the source of truth for smart-naming.

**Outcome:** single Go binary `forgectl` (alias `fx`) wrapping tmux two ways ‚Äî
- **Power mode** (laptops): typed verbs, `forgectl tmux ls`.
- **Thumb mode** (iPhone/Termius): bare `forgectl` opens a TUI menu ‚Äî zero typing, number-key select, narrow-screen layout, forgiving input (autocorrect shrugged off; unrecognized verb falls into the menu).

Go + charmbracelet, distributed via `cameronsjo/homebrew-tap`, architected so future modules (`pr`, `k8s`) slot in.

## Stack

- Go 1.23 (bumped to 1.24 by fang v1.0.0 requirement). Cobra + charmbracelet `fang` (`fang.Execute(ctx, root)`).
- TUI: Bubble Tea + Bubbles (list, textinput, viewport) + Lip Gloss + Huh.
- Distribution: standalone repo `github.com/cameronsjo/forgectl` ‚Üí goreleaser ‚Üí formula auto-published to `cameronsjo/homebrew-tap`.
- sesh delegation: pick/create + zoxide shell out to `sesh connect`/`sesh list`; everything else drives `tmux` directly.
- Name in one constant (`internal/meta.AppName`).

## Ops package (`internal/tmux`)

Exec seam (`internal/exec`): `Runner` with `Run`/`RunInteractive`. `OSRunner` prod; `FakeRunner` tests. `Client{run,tmuxBin,seshBin}`, `InsideTmux()` injectable.

Parsing: `-F` fields separated by ASCII unit-sep `\x1f`.

| Op | tmux command |
|----|---------|
| list sessions | `list-sessions -F '#{session_name}\x1f#{session_windows}\x1f#{?session_attached,1,0}\x1f#{session_created}\x1f#{session_path}'` |
| list windows (all) | `list-windows -a -F '#{session_name}\x1f#{window_index}\x1f#{window_name}\x1f#{?window_active,1,0}\x1f#{window_panes}'` |
| list panes (all) | `list-panes -a -F '‚Ä¶pane_title‚Ä¶pane_current_command‚Ä¶?pane_active‚Ä¶'` |
| exists | `has-session -t <name>` |
| rename | `rename-session -t <old> <new>` |
| kill | `kill-session -t <name>` |
| kill others | `kill-session -a -t <keep>` |
| switch (inside) | `switch-client -t <target>` |
| attach (outside) | `attach-session -t <target>` |
| last (inside) | `switch-client -l` |
| jump to window | `switch-client -t <session>:<index>` |

Load-bearing helper `AttachOrSwitch(ctx, target)`: inside ‚Üí `switch-client -t`; outside ‚Üí `attach-session -t` (interactive). `LastSession` branches on `InsideTmux()`. Types `Session{Name,Windows,Attached,Created,Path}`, `Window{Session,Index,Name,Active,Panes,Target}`, `Pane{‚Ä¶,Target}` ‚Äî Target pre-built `sess:win[.pane]`. Pick/SeshList delegate to sesh.

## Dual-mode + forgiveness

- Bare invoke ‚Üí TUI (`root.RunE` no subcommand ‚Üí `tui.Run`). `forgectl tmux` no verb ‚Üí tmux submenu.
- Aliases (Cobra-native): kill `k/rm/delete/x`; list `ls/l`; rename `mv/rn`; pick `p/go/n/new`; tree `t`; windows `w`; last `-/last`. `SuggestionsMinimumDistance=2`.
- Normalize (`internal/forgive`): lowercase, TrimSpace, TrimRight(".,!?;:").
- Unknown verb ‚Üí TUI: pre-run hook normalizes argv, `root.Find(args)`; resolves to root + non-flag ‚Üí launch TUI. Flag errors stay strict.

## TUI

Screen-router `App{client,current,width,height}`. Menu: list {Pick, Sessions, Windows, Tree, Last} number-key select. Session list: enter/number ‚Üí AttachOrSwitch; r rename, k kill (Huh), K kill-others. Window list: `<sess> ¬∑ <name>` ‚Üí JumpToWindow. Tree: viewport. Narrow (`width<60`): single-column, shrink indent, ellipsis, JoinVertical, feed width into list.SetSize.

## Testing

FakeRunner (canned stdout per argv, records calls). Targets: parsers (spaces, zero windows, detached), AttachOrSwitch branching, command construction, tree assembly, forgive.Normalize table tests, alias resolution. TUI largely untested.
